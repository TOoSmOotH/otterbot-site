Get Started

Up and running in 3 commands

01  Pull the image
$ docker pull ghcr.io/toosmooth/otterbot:latest
02  Run the container
$ docker run -d -p 62626:62626 \
  --name otterbot \
  -e OTTERBOT_DB_KEY=change-me-to-something-secret \
  --shm-size 256m \
  ghcr.io/toosmooth/otterbot:latest
03  Start chatting
$ open http://localhost:62626
# 🎉 That’s it. You’re live.
🦦

Pro tip: Set your OTTERBOT_DB_KEY to a unique secret — it encrypts your database. Add -v ~/otterbot:/otterbot if you want data to persist between container restarts. See the docs for all runtime options.
